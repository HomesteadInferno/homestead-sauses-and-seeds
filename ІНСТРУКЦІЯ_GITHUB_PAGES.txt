┌─────────────────────────────────────────────────────────┐
│ ШАГ 1: Створюємо акаунт на GitHub (якщо немає)         │
└─────────────────────────────────────────────────────────┘

1. Йдемо на https://github.com
2. Натискаємо "Sign up" (Зареєструватися)
3. Вводимо:
   - Email
   - Пароль (мінімум 8 символів)
   - Username (назва вашого профілю, наприклад "homestead-farm")
4. Підтверджуємо email


┌─────────────────────────────────────────────────────────┐
│ ШАГ 2: Створюємо репозиторій (місце для файлів)        │
└─────────────────────────────────────────────────────────┘

1. Після входу натискаємо зелену кнопку "New" (зверху зліва)
   АБО йдемо на https://github.com/new

2. Заповнюємо:
   Repository name: homestead-shop
   ⚠️ ВАЖЛИВО: Назва без пробілів, тільки англійські букви та дефіси!
   
   Description: Інтернет-магазин гострих перців та соусів
   
   ✅ Public (публічний - безкоштовно)
   ❌ НЕ ставимо галочку "Add a README file"

3. Натискаємо зелену кнопку "Create repository"


┌─────────────────────────────────────────────────────────┐
│ ШАГ 3: Завантажуємо файли сайту                         │
└─────────────────────────────────────────────────────────┘

Ти побачиш сторінку з інструкціями. Ігноруй їх!
Натискай на посилання "uploading an existing file"

1. Натискаємо "choose your files"

2. Вибираємо ВСІ файли сайту:
   ✅ index.html
   ✅ product.html
   ✅ sauces.html
   ✅ seedsandseedlings.html
   ✅ faq.html
   ✅ privacy.html          ← НОВИЙ!
   ✅ 404.html              ← НОВИЙ!
   ✅ style.css
   ✅ products.js
   ✅ cart-SECURE.js        ← НОВИЙ (замість cart.js)
   ✅ catalog.js
   ✅ product-page.js
   ✅ search.js
   ✅ search-styles.css
   ✅ robots.txt            ← НОВИЙ!
   ✅ sitemap.xml           ← НОВИЙ!
   ✅ Всі картинки (.jpg, .png)
   ✅ Всі favicon файли

⚠️ ВАЖЛИВО: Перейменуй cart-SECURE.js → cart.js перед завантаженням!

3. Прокручуємо вниз
4. В полі "Commit message" пишемо: "Initial site upload"
5. Натискаємо зелену кнопку "Commit changes"


┌─────────────────────────────────────────────────────────┐
│ ШАГ 4: Вмикаємо GitHub Pages                            │
└─────────────────────────────────────────────────────────┘

1. Натискаємо вкладку "Settings" (зверху)
2. В лівому меню натискаємо "Pages"
3. В секції "Source" вибираємо:
   Branch: main (або master)
   Folder: / (root)
4. Натискаємо "Save"

🎉 GitHub почне публікувати сайт!


┌─────────────────────────────────────────────────────────┐
│ ШАГ 5: Перевіряємо що сайт працює                       │
└─────────────────────────────────────────────────────────┘

1. Чекаємо 2-5 хвилин (GitHub створює сайт)

2. Йдемо на адресу:
   https://ВАШ-USERNAME.github.io/homestead-shop/
   
   Наприклад, якщо твій username "mykola123", то адреса:
   https://mykola123.github.io/homestead-shop/

3. Якщо сайт не відкривається:
   - Почекай ще 5 хвилин
   - Перевір чи правильно ввів адресу
   - Перевір чи в репозиторії є файл index.html


┌─────────────────────────────────────────────────────────┐
│ ШАГ 6: ОБОВ'ЯЗКОВІ ЗМІНИ В КОДІ                         │
└─────────────────────────────────────────────────────────┘

Тепер треба виправити посилання в файлах!

1. Відкрий файл robots.txt на GitHub:
   - Клікни на файл robots.txt
   - Натисни іконку олівця (Edit)
   - Знайди рядок: Sitemap: https://ВАШ-ДОМЕН.github.io/sitemap.xml
   - Заміни на: Sitemap: https://ВАШ-USERNAME.github.io/homestead-shop/sitemap.xml
   - Натисни "Commit changes" → "Commit changes"

2. Відкрий файл sitemap.xml:
   - Клікни на файл sitemap.xml
   - Натисни іконку олівця
   - Знайди всі рядки з "ВАШ-ДОМЕН"
   - Заміни на: ВАШ-USERNAME.github.io/homestead-shop
   - Натисни "Commit changes" → "Commit changes"

3. Відкрий файл privacy.html:
   - Замінь [ваш email] на реальний email
   - Замінь [ваш телефон] на реальний телефон


┌─────────────────────────────────────────────────────────┐
│ ШАГ 7: Додаємо Honeypot поле (ЗАХИСТ ВІД БОТІВ)        │
└─────────────────────────────────────────────────────────┘

Відкрий КОЖЕН HTML файл з формою замовлення:
- index.html
- product.html
- sauces.html
- seedsandseedlings.html

В кожному файлі знайди форму замовлення і додай HONEYPOT код
(дивись файл HONEYPOT_КОД.html для деталей)


┌─────────────────────────────────────────────────────────┐
│ ШАГ 8: Як оновлювати сайт в майбутньому                 │
└─────────────────────────────────────────────────────────┘

Коли треба змінити щось на сайті:

1. Йдеш на https://github.com/ВАШ-USERNAME/homestead-shop
2. Знаходиш потрібний файл
3. Натискаєш на файл
4. Натискаєш іконку олівця (Edit this file)
5. Робиш зміни
6. Прокручуєш вниз
7. Пишеш що змінив у полі "Commit message"
8. Натискаєш "Commit changes"

⏰ Зміни з'являться на сайті через 1-2 хвилини!


┌─────────────────────────────────────────────────────────┐
│ ШАГ 9: Додаємо новий товар                              │
└─────────────────────────────────────────────────────────┘

1. Завантажуєш фото товару:
   - Upload files → вибираєш фото
   - Commit changes

2. Відкриваєш products.js:
   - Клік на файл → Edit
   - Копіюєш блок товару
   - Змінюєш ID, назву, ціну, опис
   - Commit changes

3. Відкриваєш sitemap.xml:
   - Додаєш новий товар в список
   - Commit changes


┌─────────────────────────────────────────────────────────┐
│ ШАГ 10: Підключаємо свій домен (ОПЦІОНАЛЬНО)           │
└─────────────────────────────────────────────────────────┘

Якщо хочеш домен типу homestead.com замість github.io:

1. Купуєш домен на nic.ua або ukr.net (≈150 грн/рік)
2. В налаштуваннях домену додаєш CNAME запис:
   Host: www
   Value: ВАШ-USERNAME.github.io
3. В Settings → Pages → Custom domain вводиш свій домен
4. Ставиш галочку "Enforce HTTPS"

│ ШАГ 11: Безпека та моніторинг                           │
└─────────────────────────────────────────────────────────┘

✅ ЩО ВЖЕ ЗРОБЛЕНО:

⚠️ ЩО ТРЕБА ЗРОБИТИ ВРУЧНУ:

1. Google Search Console:
   - Йди на https://search.google.com/search-console
   - Додай свій сайт
   - Завантаж sitemap.xml
   - Це допоможе Google знайти твій сайт

2. Google Analytics (опціонально):
   - Йди на https://analytics.google.com
   - Створи акаунт
   - Отримай код відстеження
   - Додай його в <head> всіх HTML файлів

│ ЧАСТІ ПРОБЛЕМИ ТА РІШЕННЯ                               │

❌ Сайт не відкривається:
→ Почекай 5-10 хвилин після публікації
→ Перевір чи є файл index.html
→ Перевір Settings → Pages чи правильно налаштовано

❌ Картинки не показуються:
→ Перевір чи завантажені фото
→ Перевір чи правильні назви файлів (регістр важливий!)

❌ Пошук не працює:
→ Перевір чи підключений search.js
→ Відкрий Console (F12) і подивись на помилки

❌ Форма не відправляється:
→ Перевір Google Apps Script URL
→ Перевір чи додано honeypot поле
